# EXP-3-PROMPT-ENGINEERING-

## Aim: 
Evaluation of 2024 Prompting Tools Across Diverse AI Platforms: 
ChatGPT, Claude, Bard, Cohere Command, and Meta
Experiment:
Within a specific use case (e.g., summarizing text, answering technical questions), compare the performance, user experience, and response quality of prompting tools across these different AI platforms.

## Algorithm:
1. Define Objective & Use Case: Select a common AI evaluation task such as text summarization, technical question answering, coding assistance, or creative writing. Choose standardized input data like technical articles, programming problems, or research-based questions for consistent testing across all AI platforms.

2. Design Standardized Prompts: Create clear, detailed, and role-specific prompts to ensure fair comparison among ChatGPT, Claude, Bard/Gemini, Cohere Command, and Meta AI. Example: “You are an AI research assistant. Summarize the following technical article focusing on key concepts, applications, and findings in 250 words.”

3. Execute Prompt Testing: Run the same prompt and input data on each AI platform under similar conditions. Record generated outputs, response time, formatting quality, and interaction experience for multiple test cases.

4. Evaluate Using Performance Metrics: Assess outputs using quantitative and qualitative criteria such as accuracy, relevance, clarity, creativity, response speed, context retention, hallucination rate, consistency, and user experience. Assign performance scores on a scale of 0–100 and document strengths and weaknesses for each platform.

5. Comparative Analysis & Conclusion: Compare all AI platforms using tables, charts, graphs, and example outputs. Identify the best-performing platform for each task category, discuss limitations, recommend prompt improvements, and summarize future trends in AI prompting and generative AI systems.


## Prompt
Create a professional 10-page report titled “Evaluation of 2024 Prompting Tools Across Diverse AI Platforms” comparing ChatGPT, Claude, Bard/Gemini, Cohere Command, and Meta AI. Analyze their performance in tasks such as text summarization, technical question answering, coding, and creative writing using metrics like accuracy, response quality, speed, user experience, and prompt understanding. Include properly formatted sections such as abstract, introduction, methodology, comparative analysis, conclusion, and references with academic formatting. Add charts, tables, graphs, AI platform logos, screenshots, and visual comparisons to support the analysis. Use formal language, real-world examples, benchmark comparisons, and proper citations to make the report visually appealing, data-driven, and suitable for academic submission.
